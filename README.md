# UnlockIt

一款强大的浏览器插件，解除网页的复制、粘贴限制，支持模拟人工输入，让学习更高效！

## ✨ 核心功能

### 📝 解除复制限制
- 拦截并阻止网站的复制事件拦截
- 支持选中并复制任何文本内容

### 📋 解除粘贴限制
- 允许粘贴到被禁用的输入框
- 支持各类富文本编辑器

### ⌨️ 模拟人工输入
- **快捷键**：`Ctrl+Shift+M` 呼出浮动输入框
- **双击输入**：双击输入框快速呼出
- **速度调节**：支持慢速/正常/快速三档
- **暂停/继续**：输入过程中可随时暂停
- **进度显示**：实时显示输入进度
- 模拟真实打字效果，绕过输入限制

### 🖱️ 右键菜单
- 选中文本 → 「🔓 强制复制选中内容」
- 右键输入框 → 「🔓 强制粘贴」
- 任意位置 → 「🔌 切换插件开关」

## 📖 使用方法

### 基础使用

1. **打开控制面板** - 点击浏览器工具栏的插件图标
2. **开启总开关** - 点击「总开关」启用插件
3. **选择功能** - 根据需要开启复制、粘贴、输入功能

### 快捷键操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+M` | 切换插件开关 |
| `Ctrl+Shift+M` | 打开浮动输入框 |
| `Ctrl+C` | 复制选中内容 |
| `Ctrl+V` | 粘贴内容 |

### 模拟输入

1. 点击目标输入框（或选中）
2. 按 `Ctrl+Shift+M` 呼出浮动输入框
3. 在输入框中粘贴或输入内容
4. 选择输入速度（慢速/正常/快速）
5. 点击「开始输入」按钮 或 按 `Enter` 键
6. 插件将模拟人工打字输入内容
7. 输入过程中可随时暂停或停止


### 安全与隐私

- 插件默认关闭，不影响正常浏览
- 所有操作仅在用户主动开启后生效
- 不收集任何用户数据
- 开源透明，可审计代码


## 🐛 常见问题

**Q: 插件在某些网站不生效？**
A: 请确保插件已开启，并尝试刷新页面。部分网站可能需要特定的处理方式，欢迎反馈。

**Q: 如何恢复默认设置？**
A: 在插件控制面板中点击「重置」按钮，或在扩展管理页面中删除并重新安装插件。

**Q: 是否支持 Firefox？**
A: 当前版本主要针对 Chromium 内核浏览器（Chrome、Edge、360等），Firefox 需要适配 Manifest V2。

**Q: 浮动输入框无法输入？**
A: 请确保目标输入框已获得焦点（点击一下输入框），然后再按快捷键呼出浮动输入框。
