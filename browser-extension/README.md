# 解除复制粘贴限制 

一款强大的浏览器插件，解除网页的复制、粘贴限制，支持模拟人工输入，让学习更高效！

## ✨ 核心功能

### 📝 解除复制限制
- 拦截并阻止网站的复制事件拦截
- 支持选中并复制任何文本内容

### 📋 解除粘贴限制
- 允许粘贴到被禁用的输入框
- 支持各类富文本编辑器

### ⌨️ 模拟人工输入
- **快捷键**：`Ctrl+Shift+M` 呼出浮动输入框
- **双击输入**：双击输入框快速呼出
- 模拟真实打字效果，绕过输入限制


### 🔔 智能提示
- 插件图标徽章显示当前状态（开启时显示"ON"）
- 检测到复制限制时自动提示
- 实时状态指示器

## 🚀 安装方法

### Chrome / Edge / 360 浏览器

1. 下载插件文件夹 `browser-extension`

2. 打开浏览器扩展管理页面

3. 开启右上角的「开发者模式」

4. 点击「加载已解压的扩展程序」

5. 选择 `browser-extension` 文件夹

6. 安装成功！插件图标将显示在工具栏

### 快捷键自定义

1. 打开 `chrome://extensions/shortcuts`
2. 找到「解除复制粘贴限制」
3. 自定义快捷键：
   - **开关插件**： `Ctrl+M`
   - **打开浮动输入框**： `Ctrl+Shift+M`

## 📖 使用方法

### 基础使用

1. **打开控制面板** - 点击浏览器工具栏的插件图标
2. **开启总开关** - 点击「总开关」启用插件
3. **选择功能** - 根据需要开启复制、粘贴、输入功能

### 快捷键操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+M` | 切换插件开关 |
| `Ctrl+Shift+M` | 打开浮动输入框 |
| `Ctrl+C` | 复制选中内容 |
| `Ctrl+V` | 粘贴内容 |

### 模拟输入

1. 点击目标输入框（或选中）
2. 按 `Ctrl+Shift+M` 呼出浮动输入框
3. 在输入框中粘贴或输入内容
4. 点击「开始输入」按钮 或 按 `Enter` 键
5. 插件将模拟人工打字输入内容

### 安全与隐私

- 插件默认关闭，不影响正常浏览
- 所有操作仅在用户主动开启后生效
- 不收集任何用户数据
- 开源透明，可审计代码

## 🐛 常见问题

**Q: 插件在某些网站不生效？**
A: 请确保插件已开启，并尝试刷新页面。部分网站可能需要特定的处理方式，欢迎反馈。

**Q: 如何恢复默认设置？**
A: 在插件控制面板中点击「重置」按钮，或在扩展管理页面中删除并重新安装插件。

**Q: 是否支持 Firefox？**
A: 当前版本主要针对 Chromium 内核浏览器（Chrome、Edge、360等），Firefox 需要适配 Manifest V2。
